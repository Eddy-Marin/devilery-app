# ZoneDelivery

**ZoneDelivery** es una plataforma de delivery hiper-local enfocada en zonas desatendidas por las grandes aplicaciones.

## Documentación

La documentación del proyecto se encuentra en la carpeta `docs/`:

- [Project Charter](./docs/01_Project_Charter.md)
- [Requerimientos (SRS)](./docs/02_Requirements_SRS.md)
- [Flujos de Proceso](./docs/03_Process_Flows.md)

## Descripción

Este proyecto busca desarrollar una solución tecnológica que cubra vacíos de mercado ofreciendo tarifas competitivas y seguridad mediante validación de entrega por PIN.

## Estado del Proyecto

En fase de planificación y definición de requerimientos.
